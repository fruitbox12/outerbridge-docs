# Weave.Services Platform - Constants

This document provides a concise overview of the constants used in the Weave.Services platform. These constants are essential for maintaining consistent values across the platform, aiding in configuration, navigation, node types, and toast notifications.

## Constants Directory Structure

The `ui/src/constants` directory contains various constants that are used throughout the platform.

### Directory Structure

**Path:** `ui/src/constants`

**Files and Subdirectories:**
- `executionExample.ts`
  - Contains example constants related to execution parameters.
- `navigation.ts`
  - Contains constants related to navigation items.
- `nodes.ts`
  - Contains constants defining different node types.
- `paramsType.ts`
  - Contains constants related to parameter types.
- `toast-options.ts`
  - Contains constants for configuring toast notifications.

### Constants and Their Usage

#### `toast-options.ts`

Defines the configuration for toast notifications, which provide feedback to users.

**Example Snippet:**
```typescript
import { ToastOptions, Bounce } from 'react-toastify'

interface ToastOptionsParams {
  position?: ToastOptions['position']
  autoClose?: ToastOptions['autoClose']
  hideProgressBar?: ToastOptions['hideProgressBar']
  closeOnClick?: ToastOptions['closeOnClick']
  pauseOnHover?: ToastOptions['pauseOnHover']
  draggable?: ToastOptions['draggable']
  progress?: ToastOptions['progress']
  transition?: ToastOptions['transition']
  progressClassName?: ToastOptions['progressClassName']
  bodyClassName?: ToastOptions['bodyClassName']
  className?: ToastOptions['className']
}

export const TOAST_SUCCESSFUL_OPTION = (theme: string, options: ToastOptionsParams = {}): ToastOptions<unknown> => {
  return {
    position: options.position ?? 'bottom-left',
    autoClose: options.autoClose ?? 3000,
    hideProgressBar: options.hideProgressBar ?? false,
    closeOnClick: options.closeOnClick ?? true,
    pauseOnHover: options.pauseOnHover ?? true,
    draggable: options.draggable ?? true,
    progress: options.progress ?? undefined,
    theme: theme || 'light',
    transition: options.transition ?? Bounce,
    progressClassName: options.progressClassName,
    bodyClassName: options.bodyClassName,
    className: options.className,
  }
}
```
**Usage:**

-   **Components:** Used in components that trigger toast notifications to provide feedback to users.
-   **Modules:** Utilized in notification modules to ensure consistent toast configuration.
